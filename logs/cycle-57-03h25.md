# CYCLE 57 — VELOCITY TUNNEL VISION + TIME-OF-DAY ATMOSPHERE
**Heure** : 03h25

---

## FIXES APPLIQUES CE CYCLE

1. **Scroll velocity tunnel vision** — Quand le user scroll rapidement, les bords de l'ecran s'assombrissent proportionnellement a la vitesse. C'est un 3eme layer de vignette (v3) qui s'active avec `uScrollVelocity`. L'effet est plafonne a 25% d'assombrissement max. C'est physiquement coherent — le g-force cause du "tunnel vision" dans la realite. Ca renforce la sensation de vitesse et de chute.

2. **Time-of-day personalization** — L'experience detecte l'heure locale du user et applique un subtil shift atmospherique :
   - 5h-9h (dawn) : leger hue-rotate ambre (+10deg) — le lever du soleil sur l'horizon cosmique
   - 17h-21h (dusk) : shift chaud (+15deg) — le crepuscule
   - 21h-5h (deep night) : shift froid (-5deg) — nuit profonde
   - 9h-17h (day) : neutre — pas de modification
   Les CSS custom properties `--tod-hue` et `--tod-warmth` sont set au load. Chaque visite est donc subtilement unique — le SOTY jury ne le saura peut-etre pas consciemment, mais le SENTIRA.
