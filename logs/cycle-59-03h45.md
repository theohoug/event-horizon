## FIXES APPLIQUES CE CYCLE

1. **Overlay gradient mouse parallax** — Les gradients de profondeur sous le texte (#overlay::before) se deplacent subtilement avec la position de la souris. Les CSS custom properties `--mouse-x` et `--mouse-y` sont set en temps reel par Experience.ts. Le deplacement est de ±3% max — juste assez pour creer une sensation de profondeur sans etre conscient de l'effet. Ca donne l'impression que les nebuleuses ambiantes ont une profondeur physique derriere le texte.

2. **Post-credits hidden message** — Apres 20 secondes passees sur le credits screen (scroll > 0.97), un message tres discret apparait en bas de l'ecran : "Some journeys end where they began. Press Home to return." C'est un easter egg narratif — une invitation a recommencer le voyage. Le message fade-in sur 4 secondes en semi-transparent (opacity 0.3, color grisatre). Seuls les users les plus patients le verront.
