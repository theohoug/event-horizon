## FIXES APPLIQUES CE CYCLE

1. **Device orientation (gyroscope) support** — Sur les appareils mobiles equipes d'un gyroscope, l'inclinaison du telephone controle maintenant la position "souris". En inclinant le telephone, le user deplace la perspective de la camera, le cursor ring (invisible sur mobile mais les effets de parallax sont actifs), et les gradients de l'overlay. Le beta (avant/arriere) est mappe sur Y et le gamma (gauche/droite) sur X, avec un range de ±45 degres normalise a 0-1. La position de reference est a 45 degres de beta (l'angle naturel de tenue du telephone).

   Impact : Sur mobile, le user peut PHYSIQUEMENT explorer l'espace autour du trou noir en bougeant son telephone. Les effets de parallax texte, overlay gradients, et le text repulse reagissent au mouvement physique du device. C'est une integration hardware que tres peu de sites SOTY ont.

2. **OG Image + Twitter Image meta** — Ajout des meta tags pour l'image de partage social. L'URL pointe vers `og-image.jpg` qui devra etre cree et place dans le dossier static/. L'image sera utilisee quand le lien est partage sur Twitter, Discord, iMessage, etc.

3. **Apple Touch Icon** — Meta tag pour l'icone iOS quand le user ajoute le site en favoris.
