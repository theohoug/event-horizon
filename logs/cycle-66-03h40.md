# CYCLE 66 — RETURN TO SURFACE BUTTON
**Heure** : 03h40

---

## FIXES APPLIQUES CE CYCLE

1. **"Return to surface" button** — Un bouton discret sous le bouton "Share this journey" dans le credits screen. En cliquant, le user retourne en douceur au debut de l'experience (smooth scroll to top). Le bouton est volontairement plus petit et plus subtil que le share button — c'est une invitation, pas un call-to-action agressif.

   Style : font-mono, 0.6rem, spacing 0.12em, bordure tres discrete (8% opacity). Survol : texte et bordure s'eclairent legerement. Pas de glow, pas de shadow — c'est un whisper, pas un cri.

   Version white-mode du credits egalement supportee.

   Impact : Le user qui termine l'experience a maintenant un chemin clair pour recommencer. Avant, il fallait soit presser Home (non intuitif), soit scroller manuellement. C'est un gain d'usabilite concret pour le parcours complet.
