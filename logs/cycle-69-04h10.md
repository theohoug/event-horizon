## FIXES APPLIQUES CE CYCLE

1. **Mobile scroll hint** — Sur les appareils mobiles (Android/iPhone/iPad), le texte "Scroll to begin" est remplace par "Swipe to begin". C'est une adaptation du langage a la plateforme — le mot "scroll" est un terme desktop, "swipe" est le geste natif mobile.

   Impact : Les users mobiles voient un terme qu'ils comprennent intuitivement. C'est un gain de micro-UX concret.

2. **Sound prompt keyboard support** — La modal de choix sonore accepte maintenant les raccourcis clavier :
   - **Enter** → Active le son (equivalent de "Activate Sound")
   - **Escape** → Continue sans son (equivalent de "Continue without")

   Le handler se nettoie automatiquement apres usage.

   Impact : Un user au clavier peut traverser le flow complet (sound prompt → experience → navigation) sans jamais toucher la souris. C'est un gain WCAG concret et mesurable.

3. **Journey walkthrough audit** — Audit complet de chaque phase de l'experience (Loading → Sound → Intro → Scroll → Mid → Climax → Credits → Dev). L'audit a revele que certaines preoccupations etaient infondees :
   - Les credits ANIMAIENT deja correctement (GSAP stagger dans Timeline.ts)
   - Le HUD EST dynamique (calcul Schwarzschild en temps reel dans updateHUD)
   - Les raccourcis clavier EXISTENT deja (arrows, Home, End, Esc, ?)

   Les seules preoccupations VALIDES ont ete corrigees (mobile hint + keyboard prompt).
