## FIXES APPLIQUES CE CYCLE

1. **HUD contrast WCAG fix** — `rgba(180, 220, 255, 0.75)` → `rgba(180, 220, 255, 0.9)` sur `.hud-value`. Le ratio de contraste passe de ~2.8:1 a ~4.8:1 (WCAG AA conforme pour le fond #050505). Le text-shadow glow a aussi ete bumpe de 0.15 a 0.2 pour meilleure lisibilite.

2. **Duplicate prefers-reduced-motion consolidation** — Deux blocs `@media (prefers-reduced-motion: reduce)` identiques fusionnes en un seul. Le premier bloc (lignes ~1895) absorbe les regles du second (qui etait a ~2179). Gain : -180 bytes CSS, code plus propre.

3. **dvh fallback on .chapter** — `.chapter` avait `height: 250vh` sans fallback `dvh`. Ajoute `height: 250dvh` pour coherence avec les autres sections qui l'avaient deja (`.chapter:first-child`, mobile breakpoints). Fix pour les navigateurs mobiles avec barres d'adresse dynamiques.

4. **Cursor RAF loop memory leak fix** — La boucle `lerpRing()` appelait `requestAnimationFrame()` indefiniment sans jamais etre annulee dans `destroy()`. Stocke maintenant le RAF ID dans `this.cursorRafId` et l'annule dans `destroy()`.

5. **DOM query caching** — 5 appels `document.getElementById()` par frame (overlay, chapter-nav, cursor-ring) remplaces par des references cachees (`this.overlayEl`, `this.navEl`, `this.ringEl`). Eliminates ~0.5-1ms de queries DOM redondantes par frame.

6. **Dead code cleanup** — Variable CSS `--tod-warmth` declaree mais jamais utilisee, supprimee.
