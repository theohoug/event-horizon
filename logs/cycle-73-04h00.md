## FIXES APPLIQUES CE CYCLE

1. **Credits timeline leak fix** — Le `gsap.timeline()` dans `showCredits()` n'etait pas stocke, donc impossible a killer si l'user scroll rapidement en arriere pendant l'animation des credits. Maintenant stocke dans `this.creditsTl`, kill proprement dans `hideCredits()` et re-entree de `showCredits()`.

2. **AudioEngine disconnect cleanup** — Le `destroy()` stoppait les oscillateurs mais ne les deconnectait pas du graphe audio. Ajoute `.disconnect()` sur tous les oscillateurs, gains, filtres, compressor, et masterGain avant `ctx.close()`. Previent les references audio orphelines.

3. **dvh fallback completude** — Toutes les hauteurs de chapitres ont maintenant le fallback `dvh` :
   - `.chapter:last-child` : 300vh → 300dvh
   - `.chapter[data-chapter="3/4"]` : 280vh → 280dvh
   - `.chapter[data-chapter="7"]` : 300vh → 300dvh
   - Mobile breakpoint `.chapter:last-child` : 280dvh
   - Mobile `.chapter[data-chapter="3/4"]` : 240dvh
   - Mobile `.chapter[data-chapter="7"]` : 280dvh
