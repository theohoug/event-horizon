## FIXES APPLIQUES CE CYCLE

1. **Complete event listener lifecycle management** — Tous les `addEventListener` dans setupMouse, setupHoldInteraction, setupKeyboard, setupResize sont maintenant trackes via `addTrackedListener()`. Le `destroy()` fait un `removeEventListener` sur chacun. Elimine tout risque de memory leak par accumulation de listeners sur re-init.

2. **Interstitial breathing amplitude** — Augmentation de l'amplitude de respiration (opacity 0.85→0.75/1, letter-spacing 0.08→0.07/0.14). L'animation est plus cinematique et visible.

3. **Scroll orb glow bloom** — Remplacement de l'animation width/height (cause layout recalc) par scale transforms + glow bloom progressif. L'orbe pulse et brille en tombant. Zero layout thrashing.

4. **Focus outline 1px → 2px** — Tous les :focus-visible passent a 2px pour meilleure visibilite sur ecrans high-DPI. Accessibility win.

5. **High contrast text-stroke 0.3px → 0.5px** — Plus lisible pour les utilisateurs prefers-contrast:more.

6. **forced-colors support** — Nouveau bloc `@media (forced-colors: active)` pour Windows High Contrast Mode. Progress bar, nav dots, focus outlines, chapter text adaptes.

7. **Dead CSS removed** — `.loader-rays` et ses pseudo-elements vides supprimes (aucune reference dans le HTML).
