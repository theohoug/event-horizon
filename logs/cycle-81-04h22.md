## FIXES APPLIQUES CE CYCLE

1. **GPGPUParticles simGeo disposal** — La geometrie `PlaneGeometry(2,2)` du quad de simulation n'etait pas disposee dans `destroy()`. GPU VRAM leak corrige via `this.simQuad.geometry.dispose()`.

2. **TextReveal DOM query caching** — 5 querySelector/querySelectorAll appels PAR FRAME dans le hot path animation ont ete remplaces par des references cachees :
   - `container.querySelectorAll('.char')` → `this.chars` (deja cache depuis revealText)
   - `container.querySelector('.line.chapter-wide')` → `this.cachedSpaghettiTitle`
   - `container.querySelector('.line:not(.data):not(.emotional)')` → `this.cachedTitleLine`
   - `container.querySelector('.line.data, .line.emotional')` → `this.cachedSubtitleLine`

   Les caches sont invalides dans `revealText()` et reconstruites au prochain frame. Gain estimer : ~5-10 DOM queries supprimees par frame dans les scroll zones 0.11-0.55.

3. **BlackHole redundant multiplication** — `state.scroll * 1.0` simplifie en `state.scroll`. Code polish.
